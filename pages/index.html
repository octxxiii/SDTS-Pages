SAWS Data Transfer System

위성 기반 미사일 경보
시뮬레이션 데이터 전송

SDTS는 탄도·항공 궤적을 시뮬레이션하고 ICCS 포맷으로 UDP 패킷을 송출하는 PyQt6 기반 데스크탑 앱입니다. 실시간 모니터링, 경보/긴급 메시지, 음성 방송 등을 통해 훈련과 테스트 환경에 최적화되어 있습니다.
